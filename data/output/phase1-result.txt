====================================================================

Problem:
  `channel_server/dist/server.js` không tồn tại trong fresh clone.
  `cmd_setup()` silently skip deploy step → bridge bot không start được.
  `_cmd_doctor()` chỉ nói "not found" không có hướng dẫn cụ thể.

Fix (src/claude_bridge/cli.py):
  cmd_setup() — Step 3/4 "Channel server":
  - Nếu server.js chưa có + bun available + source root có package.json
    → tự chạy `bun run build` (subprocess.run, capture_output=True)
  - Nếu build fail → print stderr + actionable error "Run manually: cd <root> && bun run build"
  - Nếu bun không có → print error + fallback mode="mcp" + update config
  - Nếu không tìm thấy source (installed from PyPI wheel thiếu bundle)
    → hướng dẫn "pip install --upgrade claude-agent-bridge"

  _cmd_doctor() — Channel server check:
  - Nếu deployed: ✓ (hiển thị path)
  - Nếu bundled nhưng chưa deployed: ⚠ (run: bridge-cli setup)
  - Nếu không có gì: ✗ + "Fix: cd <root> && bun install && bun run build"
    (nếu source root có package.json → in đường dẫn cụ thể)

IMP-03 | install.sh hero install command
====================================================================

Problem:
  Không có one-liner install. User phải tự làm 7 bước thủ công.

Fix:
  Tạo install.sh (mới hoàn toàn) và update README.md Quick Start.

install.sh — logic:
  1. Detect OS (macOS/Linux/WSL)
  2. Check Python 3.11+ → fail with actionable message nếu thiếu
  3. Check Claude CLI → fail nếu thiếu
  4. Check/Install Bun → curl bun.sh/install nếu chưa có
  5. Check tmux → warn nếu thiếu (không blocking)
  6. Clone/update repo (git clone hoặc git pull nếu đã clone)
  7. bun install trong channel/ (nếu node_modules chưa có)
  8. bun run build → build server.js bundle
  9. pipx install -e . (primary) hoặc pip install -e . (fallback)
  10. bridge-cli doctor (health check)
  11. Print next steps

Design decisions:
  - Idempotent: git pull nếu đã clone, skip bun install nếu node_modules có
  - Fail-fast: set -e + rõ ràng từng bước
  - Actionable errors: mỗi fail() in exact command để fix
  - CLAUDE_BRIDGE_SRC env var để override install dir
  - Colors: BLUE/GREEN/YELLOW/RED cho UX tốt hơn
  - sh-compatible (không bash-specific)

README.md Quick Start:
  Thay 4-step block bằng:
    curl -fsSL https://raw.githubusercontent.com/hieutrtr/claude-bridge/main/install.sh | sh
  Giữ Manual Install section bên dưới cho ai prefer step-by-step.
