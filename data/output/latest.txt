IMP-03 | P0-3 | `install.sh` chưa tồn tại
--------------------------------------------------------------------
Mô tả:
  Packaging spec (plan/PACKAGING_SPEC.md §11) mô tả "Hero Install Command":
  `curl -fsSL https://raw.githubusercontent.com/.../install.sh | sh`
  Nhưng file này chưa được viết. User mới phải đọc 7 bước manual.

  Pattern học từ OpenClaw: `npm install -g openclaw@latest` — one-liner install.
  Bridge cần equivalent: `install.sh` download + setup tự động.

Files cần tạo/sửa:
  - install.sh                        (TẠO MỚI — theo spec §11)
  - README.md                         (Quick Start: thay 7-step bằng curl one-liner)

install.sh logic (theo spec):
  1. Detect OS (macOS/Linux/WSL)
  2. Check dependencies: python3 ≥3.11, pip/pipx, bun (hoặc hướng dẫn install)
  3. Check claude CLI in PATH
  4. pipx install claude-agent-bridge (hoặc pip install -e . nếu dev mode)
  5. cd channel && bun install && bun run build
  6. bridge-cli setup (chạy wizard)
  7. Print success + next steps

Acceptance criteria:
  [ ] `curl -fsSL <url> | sh` chạy end-to-end không cần user input (trừ setup wizard)
  [ ] Script detect thiếu dependency và print actionable error (không silent fail)
  [ ] Script idempotent: chạy lần 2 không break lần 1
  [ ] README Quick Start section dùng curl one-liner làm bước chính
  [ ] Script test trên clean Ubuntu 22.04 và macOS 14

Estimated effort: 3h
Dependencies: IMP-01, IMP-02 (setup phải work trước khi wrap vào install.sh)
Commit format: `IMP-03: add install.sh hero install command`
