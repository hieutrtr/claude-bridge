## What This Task Does
Implement bridge_dispatch, bridge_status, bridge_agents, bridge_history, bridge_kill, bridge_create_agent as real MCP tools with direct DB access. These replace the PYTHONPATH shell-outs.

## Key Architecture Constraints
- Direct DB access (no subprocess to cli.py)
- Return structured JSON strings (not CLI text)
- Error handling returns descriptive error messages
- Tools use BridgeDB from db.py

## Test Plan
Happy path:
- bridge_agents returns list of agents
- bridge_status returns running tasks
- bridge_dispatch creates task and spawns process
- bridge_history returns task list
- bridge_kill stops running task
- bridge_create_agent registers new agent

Edge cases:
- bridge_agents with no agents
- bridge_status with no running tasks
- bridge_dispatch to busy agent (queues)

Errors:
- bridge_dispatch to nonexistent agent
- bridge_kill with no running task
- bridge_create_agent duplicate name
