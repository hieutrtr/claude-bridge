- Modules: src/claude_bridge/db.py, cli.py, agent_md.py, dispatcher.py

## What This Task Does
Add model column to agents table, --model flag to create-agent and dispatch, set-model command, pass model to claude CLI.

## Key Architecture Constraints
- Default model: sonnet
- Valid models: sonnet, opus, haiku
- Agent .md frontmatter `model:` field set from config
- dispatch --model overrides agent default
- Dispatcher passes --model flag to claude CLI

## Test Plan
Happy path:
- Create agent with default model (sonnet)
- Create agent with --model opus
- Dispatch with --model override
- set-model command changes agent default
- Agent .md contains correct model field

Edge cases:
- Invalid model name rejected

Errors:
- set-model on non-existent agent returns error
