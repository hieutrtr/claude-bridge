## What This Task Does
Write the CLAUDE.md that defines how the Bridge Bot routes Telegram commands to bridge-cli.py. This file IS the Bridge Bot — Claude Code reads it and follows the routing rules.

## Key Architecture Constraints
- Bridge Bot is Claude Code session #0 with Telegram MCP
- Routes commands to `python3 -m claude_bridge.cli <command>`
- Relays output verbatim
- Natural language fallback for non-slash commands
- Never modifies projects directly

## Deliverables
- `src/claude_bridge/bridge_bot_claude_md.py` — generates the CLAUDE.md content
- `tests/test_bridge_bot.py` — validates content
- Setup instructions doc
