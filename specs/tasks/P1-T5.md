## Refs

- Modules: src/claude_bridge/cli.py
- Dependencies: P1-T1

## What This Task Does
Replace the current `setup` command (prints CLAUDE.md to stdout) with an interactive wizard that does everything in one flow: token → bot dir → channel deps → cron → print startup command.

## Key Architecture Constraints
- Existing `setup-telegram`, `setup-bot`, `setup-cron` stay as separate commands
- `setup` orchestrates them into a single wizard
- `--no-prompt` flag for scripted installs
- Idempotent: re-running skips already-done steps
- Deploys bundled server.js to ~/.claude-bridge/channel/dist/

## Test Plan
Happy path:
- setup runs all steps in order
- Creates config.json with token
- Creates CLAUDE.md + .mcp.json in bot dir
- Installs cron
- Deploys channel server

Edge cases:
- Token already exists → skip or update
- Bot dir already exists → skip or update
- Cron already installed → skip
- --no-prompt with all flags

Errors:
- Bun not installed → warn
- Invalid token format → warn but continue
