## What This Task Does
Test task dispatch: subprocess spawning with correct claude flags, PID tracking, busy rejection, SQLite state updates. All subprocess calls mocked.

## Key Architecture Constraints
- Command: `claude --agent <name> --session-id <id> --project-dir <dir> --output-format json -p "prompt"`
- stdout → workspace/tasks/task-{id}-result.json
- stderr → workspace/tasks/task-{id}-stderr.log
- `start_new_session=True` (detach from parent)
- Task status: pending → running (with PID and started_at)
- Agent state: → running
- Reject if agent already has a running task

## Test Plan
Happy path:
- spawn_task calls Popen with correct command list
- spawn_task returns PID
- cmd_dispatch creates task row, spawns process, updates state
- pid_alive returns True for running process, False for dead

Edge cases:
- Dispatch to busy agent returns error
- Dispatch to non-existent agent returns error

Errors:
- kill_process handles dead PID gracefully (ProcessLookupError)
- kill_process escalates SIGTERM → SIGKILL
