# M11.T2: Team Definition CLI Commands

## Refs
- Task def: plan/implementation/phase-3-tasks.md, Task 3.2
- Modules: src/claude_bridge/cli.py
- Dependencies: M11.T1 (teams schema + DB methods)

## What This Task Does
Add CLI commands to create, list, and delete teams: `create-team`, `list-teams`, `delete-team`.

## Key Architecture Constraints
- All referenced agents must exist
- Lead agent must not be in members list
- Error messages to stderr, output to stdout
- Exit code 0 = success, non-zero = error

## Test Plan
Happy path:
- create-team with valid agents creates team
- list-teams shows all teams with members
- delete-team removes team

Edge cases:
- Lead agent same as member (error)
- Referenced agent doesn't exist (error)
- Team with single member

Errors:
- Duplicate team name
- Delete non-existent team
- Empty team name
