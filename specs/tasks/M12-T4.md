## What This Task Does
Wire channel tracking into dispatch and history commands. Add --channel, --chat-id, --message-id flags to dispatch. History shows channel source. on-complete stores channel info for notification routing.

## Test Plan
Happy path:
- dispatch with --channel telegram --chat-id 12345 stores channel info
- history shows channel column
- team-dispatch with channel context

Edge cases:
- No channel flags defaults to 'cli'
- History filtering by channel

Errors:
- Invalid channel value
