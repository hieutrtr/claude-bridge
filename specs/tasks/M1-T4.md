## What This Task Does
Test delete-agent: validates agent exists, rejects if running task, removes .md file, removes workspace, deletes from SQLite.

## Key Architecture Constraints
- Must error if agent has a running task (user should /kill first)
- CASCADE delete removes tasks from SQLite
- Removes agent .md file from ~/.claude/agents/
- Removes workspace directory

## Test Plan
Happy path:
- Delete agent removes from DB
- Delete agent removes .md file
- Delete agent removes workspace

Edge cases:
- Delete non-existent agent returns error

Errors:
- Delete agent with running task returns error (not silent kill)

## Gaps in Existing Code
- cli.py cmd_delete_agent silently kills running tasks instead of erroring — needs fix
